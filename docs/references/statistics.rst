Statistics
==========

Statistical helpers operate on a :class:`~tempo.tsdf.TSDF` and return a new
``TSDF``. In v0.2 these are module-level functions in ``tempo.stats``; the
equivalent ``TSDF`` methods are deprecated (see :doc:`../about/user-guide`).
